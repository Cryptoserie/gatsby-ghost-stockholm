### Installation

Install and run locally:

```shell
$ git clone https://github.com/hackersandslackers/gatsby-ghost-stockholm.git
$ cd gatsby-ghost-stockholm
$ make install
$ make dev
``` 

#### Production Builds

Build and serve a Production build with `make serve`:

```shell
$ git clone https://github.com/hackersandslackers/gatsby-ghost-stockholm.git
$ cd gatsby-ghost-stockholm
$ make serve
```
